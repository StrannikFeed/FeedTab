# FeedTab

Браузерное расширение для изменения новой страницы.

## ✨ Возможности

- 📸 Drag & Drop загрузка изображений и видео
- 💾 Хранение в IndexedDB (локальное хранилище)
- 🗑️ Удаление при наведении (крестик в правом-верхнем углу)
- 🎲 Случайная картинка по клику
- 🖥️ Замена новой вкладки браузера

## 🛠️ Технологии

- **React 18**
- **IndexedDB** (idb)
- **Vite** + @crxjs/vite-plugin
- **Chrome Extensions API**
